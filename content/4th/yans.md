[Создание диаграмм для ВКР](https://github.com/morozovvaa/prog8UML/blob/main/README.md)

Используя PlantUML (в качестве расширения для IDE от JetBrains или Visual Studio Code) или ArgoUML, создать набор диаграмм, раскрывающих функционал разрабатываемого вами продукта в рамках ВКР (список тем ВКР). Диаграммы должны раскрывать динамическую и статическую сущность разрабатываемого продукта.

Необходимо реализовать следующие диаграммы:

- вариантов использования (use case); 
- классов (classes);
- последовательности (sequence); 
- диаграмма состояний (state); 
- диаграмма деятельности (activity); 

Для диаграммы классов, которая должна присутствовать среди диаграмм, сгенерировать программный код, который использовать для разработки программного решения (если применимо). Или на одном из современных языков (например, Python, Java, Typescript, Kotlin, Golang).

В презентации (отчете по выполнению задания) описать предметную область и продемонстрировать процесс генерации, описать кратко подход к выполнению задания.
